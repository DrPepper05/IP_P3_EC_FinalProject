Authentication (Public)

{
  "email": "user@example.com",

Lockers (Public Read)

GET /api/lockers/available

View all available lockers (no authentication required)

GET /api/lockers/available/size/MEDIUM

Filter available lockers by size

POST /api/lockers (Admin only) 
Authorization: Bearer {token}
{
  "lockerNumber": "L001",
  "size": "MEDIUM",
  "status": "AVAILABLE",
  "hourlyRate": 5.0
}

Bookings (Authenticated)

POST /api/bookings 
Authorization: Bearer {token}
{
  "lockerId": 1,
  "startDatetime": "2025-12-01T10:00:00",
  "endDatetime": "2025-12-01T14:00:00"
}
GET /api/bookings/my-bookings

View my bookings (customer)

PUT /api/bookings/{id}/cancel

Cancel a booking

User Management (Admin Only)

GET /api/persons

View all users

GET /api/persons/customers

View all customers
